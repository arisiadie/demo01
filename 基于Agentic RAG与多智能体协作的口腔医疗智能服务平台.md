# 基于Agentic RAG与多智能体协作的口腔医疗智能服务平台

## 一、项目简介

​		龋齿、牙周病、口腔正畸、种植修复、口腔癌筛查是高发民生医疗需求。普通患者看不懂口腔专业术语、不了解诊疗方案选择、不知道就医流程、难以辨别过度治疗、忽视用药禁忌与复诊规范；口腔科医师日常接诊量大，病历书写、患者教育、诊前问询耗时高；基层口腔机构缺乏专科辅助工具，规范化诊疗水平参差不齐。本项目依托**口腔医学临床指南、国家口腔疾病诊疗规范、口腔用药手册、口腔影像判读标准、真实诊疗病例库、口腔健康科普知识库**，采用Agentic RAG智能推理 + 多智能体协作模式，实现症状预问诊、诊疗方案解读、用药合规审查、影像辅助分析、口腔健康管理全智能化服务，是贴近大众健康需求、展示度高、答辩亮点充足的优质AI医疗项目。

## 二、四层系统架构

**口腔医学知识库 → Agentic RAG智能引擎 → 五大口腔智能体 → 口腔医疗服务前端平台**

### 1. 数据层

口腔疾病临床诊疗指南（龋病、牙周病、错颌畸形、口腔黏膜病、牙髓病等）、国家口腔疾病诊疗规范、口腔科常用药物手册（抗菌药物、局麻药物、漱口水用药规范）、正畸/种植/修复治疗方案标准、口腔X线与CBCT影像判读规范、口腔疾病典型病例库、儿童口腔保健手册、口腔术后护理与复诊指引、口腔医疗纠纷典型案例。

### 2. Agentic RAG引擎

口腔需求意图识别（症状咨询、方案解读、用药查询、复诊指导、健康科普）、复合口腔问题拆解（多牙位 + 多症状 + 多病史交叉）、多轮指南与病例检索、诊疗方案合规自检、用药禁忌溯源校验、诊断结论纠错与依据标注，有效抑制AI在医疗场景下的幻觉风险，确保所有输出可溯源至权威口腔临床指南。

### 3. 五大口腔智能体

**智能体1：口腔症状预问诊智能体**

根据患者主诉（牙痛、牙龈出血、牙齿松动、口腔溃疡、颌面肿胀等）、持续时长、伴随症状、既往病史，智能分析可能的口腔疾病类型（龋病/牙髓炎/牙周炎/冠周炎/黏膜病等），评估就诊紧急程度，生成结构化预问诊报告，辅助接诊医师快速掌握患者病情概况，同时向患者通俗解释可能病因与建议就诊科室。

**智能体2：诊疗方案解读智能体**

针对患者已获得的诊断结论与治疗方案（根管治疗、种植牙、正畸矫正、牙周洁治、烤瓷冠修复等），通俗解读治疗步骤、疗程安排、费用构成、方案优劣对比、替代方案说明，帮助患者真正理解自己的治疗计划，规避因信息不对称导致的过度治疗与医疗纠纷，同时为基层诊所医师提供标准方案参考。

**智能体3：口腔用药合规审查智能体**

根据患者用药方案（抗生素、止痛药、漱口水、局麻药、牙周辅助药物等），结合患者年龄、妊娠状态、基础疾病（糖尿病、心脏病、凝血异常等）、过敏史，智能核查用药禁忌、剂量合规性、药物相互作用风险、儿童/老人特殊用药安全，标注风险点并输出合规用药建议，辅助医师规避用药差错。

**智能体4：口腔影像辅助分析智能体**

支持上传口腔全景片、根尖片、CBCT影像描述或报告文本，结合影像判读规范，智能辅助识别龋坏位置与深度、根尖周病变、骨吸收程度、阻生齿方位、种植位点骨量评估等关键信息；对影像报告进行通俗化解读，帮助患者理解影像结果；同时为医学生与基层医师提供影像知识辅助学习支持（注：本平台输出仅作辅助参考，不替代执业医师诊断）。

**智能体5：口腔健康管理与科普智能体**

根据用户年龄段（儿童乳牙期、青少年换牙期、成人、老年人）、既往口腔病史、治疗记录，智能生成个性化口腔健康管理方案：刷牙方式指导（巴氏刷牙法）、牙线/冲牙器使用规范、窝沟封闭建议、定期洁牙周期提醒、正畸复诊计划、种植体维护要点；同时定期推送口腔健康知识，实现从就诊到居家管理的全周期口腔健康服务闭环。

### 4. 前端层

智能预问诊入口、诊疗方案查询与解读、用药安全核查、影像报告上传分析、口腔健康档案管理、科普知识推送、就诊记录与复诊提醒、历史咨询记录。

## 三、核心业务流程

用户输入口腔症状/上传影像报告/查询用药方案 → 系统意图识别（预问诊/方案解读/用药审查/影像分析/健康管理）→ Agentic RAG拆解多维度口腔需求 → 多轮检索口腔临床指南、用药手册、病例库 → 调度对应口腔智能体执行任务 → 生成预问诊报告/方案解读/用药风险提示/影像解读/健康计划 → 前端展示归档，保留就诊与复诊记录。

## 四、技术栈

- **后端框架**：FastAPI 
- **大模型**：deepseekv4pro
- **向量数据库**：Chroma 
- **RAG核心**：LangChain（快速实现Agentic智能逻辑）
- **前端**：html+css+js
- **数据库**：MySQL
- **版本管理工具**：git

## 五、项目创新点

1. **技术创新**：突破传统口腔问答系统单轮检索局限，采用Agentic自主推理架构，对多症状、多牙位、多病史交叉的复杂口腔问题进行智能拆解、多轮检索、自我合规校验，大幅提升专业性与安全性。

2. **架构创新**：采用五智能体岗位化分工，完全贴合口腔医疗场景中预问诊、方案解读、用药审查、影像分析、健康管理的真实业务分工，模块化程度高、结构清晰规范。

3. **业务创新**：覆盖就诊前（症状预判）→ 就诊中（方案/用药/影像辅助）→ 就诊后（健康管理/复诊提醒）全周期口腔服务闭环，解决患者信息不对称、基层诊所规范化不足两大核心痛点。

4. **安全创新**：所有输出内容均基于权威口腔临床指南与真实病例，可溯源、可校验，内置医疗安全免责机制，明确区分AI辅助参考与执业医师诊断边界，抑制AI幻觉在医疗场景下的安全风险。

## 六、功能分级

**保底**：口腔疾病知识问答、常见症状科普解读、基础用药查询、简单健康建议生成、前端信息展示与历史记录管理。

**拔高**：Agentic多轮推理与复合口腔问题拆解、结构化预问诊报告生成、诊疗方案多维度对比解读、用药禁忌智能合规审查、影像报告文本辅助分析与通俗化解读、个性化口腔健康管理方案生成、多智能体协同调度、临床指南溯源自检机制、复诊提醒与健康档案全周期管理。
